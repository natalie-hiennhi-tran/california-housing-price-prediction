# California Housing Price Prediction

## Overview
This project analyzes California housing prices using data from the 1990 U.S. Census. The goal is to identify key economic and geographic factors influencing housing values and to compare different regression model specifications.

## Data
- Source: 1990 U.S. Census (California Housing dataset)
- Observations: 20,640
- Response variable: Median house value

## Methods
- Exploratory data analysis (EDA)
- Linear, log-linear, and double-log regression models
- Interpretation of economic and geographic effects

## Key Findings
- Median income is the strongest predictor of housing prices
- Coastal proximity is associated with higher housing values
- The double-log model provides the best overall fit
